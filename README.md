# PhotoMarket - Photography Portfolio Monetization Platform

A blockchain-based photography portfolio monetization and visual content rewards platform built on Stacks, enabling photographers to monetize their visual artistry through transparent appreciation metrics and fair compensation.

## Overview

PhotoMarket provides photographers with a decentralized platform to showcase and monetize their portfolios through visual appreciation engagement while ensuring transparent and equitable reward distribution based on artistic contributions.

## Features

- Portfolio content uploading with photography style verification
- Featured photography style management system
- Visual appreciation reward calculation and distribution
- Transparent photographer portfolio tracking and monetization
- Gallery curator oversight and governance

## Smart Contract Functions

### Public Functions
- `establish-photo-market`: Initialize photography portfolio monetization platform
- `feature-photography-style`: Feature photography styles for portfolio monetization
- `upload-portfolio-content`: Upload portfolio content with photography style
- `curate-appreciation-rewards`: Curate visual appreciation rewards
- `monetize-photography-portfolio`: Monetize portfolio and claim rewards

### Read-Only Functions
- `get-photographer-portfolio`: Get photographer's total portfolio tokens
- `get-photography-style`: Get photographer's photography style
- `get-total-photo-tokens`: Get total photo tokens
- `is-style-featured`: Check photography style featured status

## Usage

Deploy the contract and initialize with a gallery curator. Feature photography styles, then photographers can upload portfolios and monetize content based on their contributions.

## Security

- Gallery curator authorization controls
- Photography style featuring system for verified monetization
- Input validation for all portfolio uploading entries
- Portfolio verification before reward distribution
